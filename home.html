The Structure of an HTML Document

A standardized hierarchy of nested elements that tells a web browser how to interpret and display a webpage.

Basic Code

Every standard HTML5 document follows this fundamental pattern:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Main Heading</h1>
    <p>This is where your content goes.</p>
</body>
</html>

Core Components Explained

  • <!DOCTYPE html>: Informs the browser the document is HTML5.
  • <html>: The root element wrapping all content.
  • <head>: Contains metadata like character encoding and page titles.
  • <body>: Contains all visible content like text and images.

Modern Semantic Layout Elements

  • <header>: Introductory content or navigation.
  • <nav>: Navigation links.
  • <main>: The unique, primary content.
  • <footer>: Bottom information (copyright, contact info).
HTML Structure in Tamil

HTML ஆவணத்தின் அமைப்பு (Structure of HTML)

ஒரு இணையதளத்தை பிரவுசர் எவ்வாறு புரிந்துகொண்டு திரையில் காட்ட வேண்டும் என்று சொல்லும் ஒரு நிலையான படிநிலை அமைப்பு இதுவாகும்.

அடிப்படை குறியீடு (Basic Code)

ஒவ்வொரு நிலையான HTML5 ஆவணமும் இந்த அடிப்படை மாதிரியைப் பின்பற்றும்:

<!DOCTYPE html>
<html lang="ta">
<head>
    <meta charset="UTF-8">
    <title>பக்கத்தின் தலைப்பு</title>
</head>
<body>
    <h1>முக்கிய தலைப்பு</h1>
    <p>இங்குதான் உங்கள் தகவல்கள் இடம்பெறும்.</p>
</body>
</html>

முக்கிய கூறுகள் விளக்கம் (Core Components Explained)

  • <!DOCTYPE html>: இது ஒரு HTML5 ஆவணம் என்பதை பிரவுசருக்குத் தெரிவிக்கிறது.
  • <html>: அனைத்து உள்ளடக்கங்களையும் உள்ளடக்கிய வேர் (Root) உறுப்பு.
  • <head>: மெட்டாடேட்டா, எழுத்துரு குறியீட்டு முறை மற்றும் தலைப்பு ஆகியவற்றைக் கொண்டிருக்கும்.
  • <body>: திரையில் தெரியக்கூடிய உரை மற்றும் படங்கள் போன்ற அனைத்து உள்ளடக்கங்களையும் கொண்டிருக்கும்.

நவீன அமைப்புக் கூறுகள் (Modern Semantic Layout Elements)

  • <header>: அறிமுக உள்ளடக்கம் அல்லது மெனு (Navigation).
  • <nav>: வழிசெலுத்தல் இணைப்புகள் (Links).
  • <main>: பக்கத்தின் முக்கியமான மற்றும் தனித்துவமான உள்ளடக்கம்.
  • <footer>: பதிப்புரிமை மற்றும் தொடர்பு விவரங்கள் போன்ற அடிக்குறிப்புத் தகவல்கள்.
