The Structure of an HTML Document

A standardized hierarchy of nested elements that tells a web browser how to interpret and display a webpage.

Basic Code

Every standard HTML5 document follows this fundamental pattern:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Main Heading</h1>
    <p>This is where your content goes.</p>
</body>
</html>

Core Components Explained

  • <!DOCTYPE html>: Informs the browser the document is HTML5.
  • <html>: The root element wrapping all content.
  • <head>: Contains metadata like character encoding and page titles.
  • <body>: Contains all visible content like text and images.

Modern Semantic Layout Elements

  • <header>: Introductory content or navigation.
  • <nav>: Navigation links.
  • <main>: The unique, primary content.
  • <footer>: Bottom information (copyright, contact info).
